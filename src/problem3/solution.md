## Solution
**1. Incorrect property usage**
   **WalletBalance** interface doesnt contain a **blockchain** but **getPriority(balance.blockchain)** *(balance defined WalletBalance first)* call **blockchain**.
   This will cause runtime isssues.
   **Solution**: Add blockchain string to the WalletBalance interface
   ```
    interface WalletBalance {
    currency: string;
    amount: number;
    blockchain: string;
    }
   ```
**2. Filtering logic issue**
